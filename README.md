# Online Customer Booking System (Final Year Project)

This is a private repository created for interview and evaluation purposes.

## Overview
A web-based booking system developed as a Final Year Project using MVC architecture.

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- MVC Architecture

## Features
- Online customer booking
- Admin management (CRUD)
- Database-driven system

## Setup
1. Copy config.example.php and rename it to config.local.php
2. Update database credentials
3. Import database SQL file
4. Run on local server (XAMPP)

## Note
Sensitive configuration files are excluded from version control.
